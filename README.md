# Box Shooter Game 🎮

A simple **browser-based 2D shooting game** built using **HTML, CSS, and JavaScript**.
The player controls a blue box, collects coins, avoids enemies, and shoots bullets.

The game works on **both PC and mobile browsers** and includes a **mobile analog joystick for movement**.

---

## Features 🚀

* Player movement
* PUBG-style **analog joystick for mobile**
* Bullet shooting system
* Enemy that chases the player
* Collectible coins
* Health system
* Score tracking
* Works on **desktop and mobile browsers**
* Built using **HTML5 Canvas**

---

## Controls 🎮

### PC Controls

* Use the **mouse click on Shoot button** to fire bullets.
* Player movement can be controlled using the **joystick on screen**.

### Mobile Controls

* Use the **analog joystick** (bottom left) to move.
* Tap the **shoot button** (bottom right) to fire.

---

## Game Objects 🧩

| Object          | Description              |
| --------------- | ------------------------ |
| 🔵 Blue Box     | Player                   |
| 🟡 Yellow Boxes | Coins to collect         |
| 🟠 Orange Box   | Enemy chasing the player |
| ⚪ White Bullets | Player's weapon          |

---

## Objective 🏆

* Collect **yellow coins** to increase your score.
* Avoid the **orange enemy** or you will lose health.
* Shoot and survive as long as possible.

## How to Run the Game ▶️

1. Download the project files.
2. Save the main file as:

```
game.html
```

3. Open the file in any browser:

* Google Chrome
* Firefox
* Edge
* Mobile Browser

No installation is required.
